Thank you for purchasing this template. This document will help you customize the template easily.

1. File Structure

index.html
style.css
script.js
assets/
   images/
documentation/

2. Changing Logo

Open index.html and find:

<div class="logo">KALAKRITI</div>

Replace KALAKRITI with your brand name.

3. Changing Hero Image

Open style.css and find:

.hero { background: url('assets/images/hero.jpg'); }

Replace hero.jpg with your own image inside assets/images folder.

4. Editing Products

Open script.js.

Inside the array:

const featuredProducts = [ ... ]

You can change product name, price, and image path.

5. Changing Colors

Open style.css and edit color variables inside :root.

6. Image Credits

Images used are sourced from free commercial-use platforms. Replace with your own images for production use.
