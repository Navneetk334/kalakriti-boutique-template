KALAKRITI – Ecommerce Boutique Template
=======================================

Thank you for purchasing this template.

---------------------------------------
1. FILE STRUCTURE
---------------------------------------

index.html       → Main website file
style.css        → All styling and layout
script.js        → Product data and cart logic
assets/images/   → All website images
documentation/   → HTML documentation file

---------------------------------------
2. CHANGING LOGO
---------------------------------------

Open index.html

Find:
<div class="logo">KALAKRITI</div>

Replace KALAKRITI with your brand name.

---------------------------------------
3. CHANGING HERO IMAGE
---------------------------------------

Open style.css

Find the .hero section background image:

url('assets/images/hero.jpg')

Replace hero.jpg with your own image
inside the assets/images folder.

---------------------------------------
4. EDITING PRODUCTS
---------------------------------------

Open script.js

Inside:

const featuredProducts = [ ... ]
const bestSellers = [ ... ]

You can change:
- Product name
- Price
- Image path

Make sure your images are placed in:
assets/images/

---------------------------------------
5. CHANGING COLORS
---------------------------------------

Open style.css

Edit color variables inside:

:root { }

You can modify:
--primary
--secondary
--accent

---------------------------------------
6. IMAGE CREDITS
---------------------------------------

Images used in demo are from free commercial-use platforms.
Replace with your own images for production use.
